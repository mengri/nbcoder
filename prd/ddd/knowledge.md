# 知识库领域（Knowledge Domain）需求说明

## 领域职责
- 负责知识库文档的管理、检索、RAG 上下文注入。
- 支持文档上传、索引、分片、版本管理。

## 主要实体
- Document（文档）
- DocumentIndex（文档索引）
- Chunk（文档分片）

## 主要功能需求
1. 支持 Markdown 文档上传、目录管理、版本控制。
2. 支持全文检索、分片索引、上下文召回。
3. 支持知识库片段注入 Agent 上下文。
4. 支持文档变更追溯与血缘关系。

## 限界上下文
- 仅管理知识库文档及索引，不涉及需求卡片、任务、代码。
- 与 Requirement、Agent 领域通过上下文注入解耦。
