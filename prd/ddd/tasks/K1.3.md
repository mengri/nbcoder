# 任务：K1.3 全文检索与上下文召回

## 用户故事
作为一名终端用户，我希望能够对知识库进行全文检索，并能召回相关上下文片段，提升知识利用效率。

## 领域上下文
- 所属领域：知识库领域（Knowledge Base Domain）
- 领域职责：全文检索、上下文召回、与Agent集成。
- 相关聚合根/实体：Index, DocumentChunk
- 依赖任务：K1.2

## 任务目标
实现知识库的全文检索与上下文召回功能。

## 输入
- 领域建模说明（见 prd/ddd/knowledge.md）
- 检索与召回规则

## 输出
- 全文检索与上下文召回实现代码

## 实现要点
- 支持多种检索方式（关键词、语义等）
- 检索结果可追溯原文档
- 支持与Agent上下文集成

## 交付标准
- 代码可独立运行，单元测试覆盖主要检索与召回流程
- 检索方式与领域建模一致
