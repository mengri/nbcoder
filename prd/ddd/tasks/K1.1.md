# 任务：K1.1 文档上传与目录管理

## 用户故事
作为一名知识库管理员，我希望能够上传文档并进行目录管理，便于知识的结构化存储和后续检索。

## 领域上下文
- 所属领域：知识库领域（Knowledge Base Domain）
- 领域职责：文档管理、目录结构、与Agent集成。
- 相关聚合根/实体：Document, Directory
- 依赖任务：无

## 任务目标
实现文档上传、目录创建、文档与目录的关联管理。

## 输入
- 领域建模说明（见 prd/ddd/knowledge.md）
- 文档元数据、目录结构

## 输出
- 文档上传与目录管理实现代码

## 实现要点
- 支持多级目录结构
- 文档与目录关联可追溯
- 支持批量上传与管理

## 交付标准
- 代码可独立运行，单元测试覆盖主要上传与管理流程
- 目录结构与领域建模一致
